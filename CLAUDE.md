# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Что это за репозиторий

Это **фабрика сайтов** + первый собранный сайт. Репозиторий совмещает две роли:

1. **Переиспользуемый агент** («фабрика») — собирает статические сайты-лендинги под разных клиентов:
   - навык: `.claude/skills/build-site/SKILL.md` (вызывается командой `/build-site`)
   - шаблоны блоков: `templates/`
   - правила и стек: `docs/conventions.md`
   - публикация: `docs/deploy.md`
2. **Выход** — собранный сайт в корне (`index.html`, `css/`, `js/`) + брифы клиентов в `projects/`.

Чтобы перенести агента в новый репозиторий сайта — скопируй `.claude/skills/build-site/`, `templates/` и `docs/`.

## Перед началом работы

1. Прочитай `docs/conventions.md` — стек, структура, правила вёрстки, чек-лист качества.
2. Если строишь сайт — прочитай бриф клиента в `projects/<client>/brief.md`. **Без заполненного брифа сайт не собирается** — сначала заполни анкету по `docs/brief.template.md`.
3. Запуск сборки — навык `/build-site`.

## Стек (кратко)

Чистые HTML5 / CSS3 / Vanilla JS, без сборщиков и зависимостей. Хостинг — GitHub Pages. Подробности и причины — в `docs/conventions.md`.

## Telegram-бот брифа

`brief-bot/` — Cloudflare Worker: Telegram-бот, через который клиент пошагово заполняет бриф (контакты, ниша, ссылки, опыт, конкуренты, контент, рекламные каналы с бюджетами за 3 месяца). Фиксирует согласие на обработку ПДн, позволяет вернуться к любому шагу (`/edit`), напоминает через 2 и 24 часа о незавершённом брифе (в рабочее время по часовому поясу клиента, пояс — из вопроса про город). Готовый бриф уходит письмом на почту (Excel-файл во вложении, через Cloudflare Email Routing), а менеджеру в Telegram приходит уведомление «Бриф заполнен». Развёртывание — `brief-bot/README.md`.

## Текущий проект

`projects/klac/` — сайт маркетингового агентства **К.Л.А.Ц** (Креатив · Лидогенерация · Автоматизация · Цифровизация). Цель сайта — сбор заявок. Услуги: таргет, контекст, анализ продаж, анализ конкурентов.
